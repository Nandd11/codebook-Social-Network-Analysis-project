# FEATURES

## ✅ Data Cleaning
- Loads raw JSON data
- Normalizes user schema: id, name, friends, pages
- Removes duplicates and handles missing values
- Exports cleaned dataset

## ✅ Social Network Analysis (NetworkX)

- Degree centrality
- Betweenness centrality
- PageRank
- Connected components
- Community detection (greedy modularity)

### 1) People You May Know
- Mutual friends based ranking

### 2) Pages You Might Like
- Interest overlap based ranking
